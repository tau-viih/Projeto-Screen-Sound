# 🎧 Screen Sound API
Sistema back-end para gerenciamento de músicas, artistas e bandas, inspirado em serviços de streaming.
## Status do projeto: 🚧 Em desenvolvimento.
---

## 📌 Sobre o projeto

O **Screen Sound** é uma aplicação desenvolvida em C# com foco em back-end, responsável por gerenciar dados de um sistema de músicas, funcionando como a "engrenagem" por trás de um aplicativo de streaming.

Este projeto foi desenvolvido como parte de um trabalho acadêmico, com o objetivo de aplicar conceitos de desenvolvimento de software e versionamento com GitHub.

---

## 🎯 Problema

Hoje, muitos sistemas de música são complexos ou fechados, dificultando o entendimento de como funciona a lógica por trás de um serviço de streaming.

Este projeto busca resolver isso criando um sistema simples e funcional para:

- Cadastro de músicas
- Gerenciamento de artistas e bandas
- Avaliação de bandas

---

## 💡 Solução

Uma API back-end desenvolvida em C# que permite:

- Armazenar dados de músicas, artistas e bandas
- Realizar operações de CRUD (Create, Read, Update, Delete)
- Avaliar bandas
- Organizar informações de forma estruturada

---

## 🚀 Funcionalidades

- 🎵 Cadastro de músicas
- 👩‍🎤 Cadastro de artistas e bandas
- ⭐ Avaliação de bandas
- 📋 Listagem de dados
- ✏️ Edição de informações
- ❌ Remoção de registros
